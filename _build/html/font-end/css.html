1. CSS¶

less和sass均是css预处理器，即将带有变量、嵌套、混入、继承等编程语法的css文件解析为常规的css文件。

  • css不是编程语言，而sass是
  • css编程化 和 js模块化 均是前端代码越来越庞大、复杂的情况下对常规css和js的编程化改进

1.1. less¶

Leaner Style Sheets。在css的基础上增加了变量、嵌套、混入、继承等语法

1.2. sass¶

Syntactically Awesome Stylesheets。在less的基础上增加了条件、循环等语句
